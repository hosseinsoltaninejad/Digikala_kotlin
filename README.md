# Android-MVVM-RxJava-Digikala_kotlin
Digikall sample app developed MVVM architecture design pattern that follow the best practices of Object Oriented Design Patterns with following technology stack.

✅ Architecture Design Pattern <br/>
✅ MVVM <br/>
✅ Koin (Dependency Injection) <br/>
✅ Live Data <br/>
✅ RxJava <br/>
✅ Retrofit <br/>
✅ Repository Pattern <br/>
✅ AndroidX <br/>
✅ JetPack Libraries <br/>
✅ ...
